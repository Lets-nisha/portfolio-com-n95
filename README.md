# Portfolio Website - React

## Overview
Responsive Portfolio website built using React and Tailwind CSS.

## Tech Stack
- React
- Tailwind CSS
- React Router
- Vercel Deployment

## Pages
- Home
- About
- Services
- Project
- Skills
- Contact

## Run Locally
npm install
npm run dev

## Deployment
Deployed using Vercel by connecting GitHub repository.
